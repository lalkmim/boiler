# boiler-pes3

This is a boilerplate for a NodeJS web application using [Passport](http://passportjs.org), [ExpressJS](http://expressjs.com), [Socket.IO](http://socket.io), [Sass](http://sass-lang.com) and [Sequelize](http://sequelizejs.com) (PES3), using SQLite as database. The design is based on Arcana template from HTML5 UP (http://html5up.net/arcana).
